JcMeasurementsService

public class JcMeasurementsService : JcConnectionRequest<JcCumulocityConnection>

Allows measurements to be fetched and posted to Cumulocity. Leverages the underlying api at endpoint ‘\/measurement/measurements’ and described here

  • pageSize

    Used when fetching JcMeasurements to determines the maximum number allowed in a single request, default is 50

  • get(forSource:pageNum:from:to:reverseDateOrder:completionHandler:)

    Returns a page JcPagedMeasurements of JcMeasurement instances generated by a device for the given internal c8y id.

    Declaration

    Swift

    public func get(forSource id: String, pageNum: Int, from: Date, to: Date, reverseDateOrder: Bool, completionHandler: @escaping (JcRequestResponse<JcPagedMeasurements>) -> Void) -> URLSessionDataTask

    Parameters

    forSource

    internal id of the JcManagedObject that is the source of measurements to be returned
    pageNum

    The page to be fetched, total pages can be found in via the statistics property statistics defined by JcPageStatistics of the returned JcPagedMeasurements object
    from

    Date and time to search from
    to

    Date and time to search upto (use Date() to specify up to now)
    reverseDateOrder

    if false results are ordered date descending, specify true (default) for the opposite i.e. the last measurement will be first in the list
    completionHandler

    callback function which will be called with the results

    Return Value

    task thread of http request
